## Weekend Class Code

Repositori ini berisi kumpulan materi dan contoh kode yang digunakan dalam kelas **Superteam Indonesia Weekend Class**. Semua contoh di sini dimaksudkan sebagai referensi belajar, eksperimen, dan latihan mandiri peserta.

### Tentang Repositori Ini

- **Fokus utama**: Latihan pemrograman dan eksperimen kode yang dibahas selama sesi kelas.
- **Isi**: Contoh kode, snippet, dan proyek kecil yang menunjukkan konsep yang diajarkan.
- **Tujuan**: Membantu peserta memahami konsep secara praktis melalui kode yang bisa dijalankan dan dimodifikasi sendiri.

Silakan gunakan repositori ini sebagai tempat:

- Mempelajari kembali materi yang sudah dibahas di kelas.
- Mencoba variasi atau eksperimen dari contoh yang ada.
- Menyimpan solusi latihan versi Anda sendiri.

### Kebutuhan Instalasi

Untuk dapat menjalankan kode di repositori ini, disarankan memiliki:

- **Git**: Untuk meng-clone dan mengelola versi kode.
- **Node.js (versi LTS terbaru)**: Banyak contoh modern menggunakan ekosistem JavaScript/TypeScript.
- **npm atau yarn**: Untuk meng-install dependensi proyek (jika ada `package.json`).
- **Rust toolchain (rustup + cargo)**: Digunakan untuk meng-compile program Rust dan smart contract.
- **Solana CLI**: Untuk berinteraksi dengan cluster Solana (localnet, devnet, dll).
- **Anchor CLI**: Framework untuk mengembangkan dan menguji Solana program dengan lebih mudah.
- **Code editor** (disarankan: Visual Studio Code atau Cursor): Untuk menulis dan membaca kode dengan nyaman.

Pastikan juga:

- Sistem operasi Anda sudah mendukung Node.js, Rust, dan Solana CLI (Windows, macOS, atau Linux).
- Anda memiliki akses ke terminal/command prompt untuk menjalankan perintah `node`, `npm`, `cargo`, `solana`, atau `git`.

### Cara Mulai Menggunakan

1. Clone repositori ini:

   ```bash
   git clone <URL-REPO-INI>
   cd weekend-class-code
   ```

2. Jika ada proyek dengan `package.json`, install dependensinya:

   ```bash
   npm install
   # atau
   yarn install
   ```

3. Ikuti instruksi spesifik di masing-masing folder/proyek (jika tersedia `README` tambahan).

### Kontribusi dan Penggunaan

- **Peserta kelas** dipersilakan untuk:
  - Menambahkan folder atau file latihan sendiri.
  - Mencoba memodifikasi contoh yang sudah ada.
- Jika repositori ini dibagikan sebagai referensi publik, gunakan hanya untuk tujuan belajar dan jangan menyimpan data sensitif/rahasia.

Selamat belajar dan bereksperimen dengan kode!
